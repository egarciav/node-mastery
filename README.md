# Node.js Mastery 🚀

Sitio de aprendizaje para dominar Node.js desde cero hasta nivel avanzado. Construido como referencia práctica con ejemplos de código reales y explicaciones claras.

## ¿Qué encontrarás aquí?

Todo lo que necesitas para trabajar con Node.js en 2026:

- **Fundamentos** — Event Loop, módulos, async/await, streams, File System
- **Express** — Routing, middleware, manejo de errores, archivos estáticos
- **APIs REST** — CRUD, autenticación JWT, autorización RBAC, validación con Zod
- **Bases de datos** — MongoDB con Mongoose, PostgreSQL con Prisma, migraciones
- **Testing** — Vitest, Supertest, mocks, tests de integración y E2E
- **NestJS** — Controllers, services, módulos, pipes, guards
- **DevOps** — Docker, CI/CD con GitHub Actions, PM2, logging con Pino
- **Node Moderno** — fetch nativo, ESM, Permission Model (Node 20+)
- **Arquitectura** — Clean Architecture, patrones Repository/Service, SOLID

## Correr en local

```bash
npm install
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173).
